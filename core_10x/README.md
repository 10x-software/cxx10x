# py10x-kernel

10X Platform Core Kernel is C++ extension with Python bindings for traitables, traits, nodes, and utilities.

This package is a dependency of `py10x-core` and is intended to be used in conjunction with [py10x-core](https://github.com/10x-software/py10x/blob/main/README.md) only.

## Requirements

- Python 3.11 or 3.12
- Modern windows, macOS, or Linux
